# AI Book — 从基础到实践，系统掌握人工智能

欢迎来到 AI Book！这是一本系统性的 AI 知识库，涵盖以下四大主题：

- **机器学习基础** — 监督学习、无监督学习、模型评估与特征工程
- **大语言模型** — Transformer 架构、预训练与微调、推理优化
- **提示工程** — Chain-of-Thought、Few-shot、Agent 提示模式
- **AI 应用开发** — RAG、LangChain、AI Agent 开发实战

## 如何使用本书

本书采用由浅入深的结构，建议按章节顺序阅读。每个主题都包含：

1. **概述** — 快速了解该领域的核心概念
2. **原理** — 深入理解背后的技术细节
3. **实践** — 通过示例和代码动手练习

选择左侧导航栏中的主题，开始你的学习之旅！
