# AI Book — 从基础到实践，系统掌握人工智能

欢迎来到 AI Book！这是一本系统性的 AI 知识库，围绕四大核心主题组织：

## [大语言模型](/docs/llm/overview)

从机器学习基础到 LLM 前沿应用，完整覆盖大语言模型的知识体系：

- **基础概念** — 学习范式、过拟合与欠拟合、模型评估、特征工程
- **Transformer 架构** — 自注意力机制、多头注意力、位置编码
- **预训练与微调** — 数据准备、LoRA/QLoRA、RLHF/DPO、评估基准
- **主流模型** — GPT、LLaMA、Mistral、Qwen、DeepSeek
- **推理优化** — KV Cache、量化、vLLM、投机解码
- **提示工程** — Zero/Few-shot、Chain-of-Thought、角色扮演
- **RAG** — 检索增强生成、向量数据库、混合检索
- **AI Agent** — ReAct、规划模式、多 Agent 协作
- **应用开发框架** — LangChain、LlamaIndex、可观测性与评估

## [Llama.cpp 实现原理](/docs/llama-cpp/index)

深入 llama.cpp 源码，理解 C/C++ 推理引擎的内部实现：

- 项目架构与模块划分
- 启动流程与线程模型
- REST API 与 C 库接口
- 核心数据结构（ggml_tensor、llama_model 等）
- GGUF 存储格式与加载机制
- 通信协议与采样器链
- 后端抽象与硬件加速

## [SGLang 实现原理](/docs/sglang/index)

深入 SGLang 源码，理解 Python LLM 服务框架的内部实现：

- 多进程架构（HTTP 服务器、调度器、解码器）
- 基数树 KV 缓存与自动前缀共享
- 张量并行 / 流水线并行 / 数据并行
- OpenAI 兼容 HTTP API 与扩展端点
- 核心数据结构与内存池管理
- ZMQ IPC 通信协议

## [vLLM 实现原理](/docs/vllm/index)

深入 vLLM 源码，理解高吞吐量 LLM 服务引擎的内部实现：

- PagedAttention 与块级 KV 缓存管理
- 连续批处理与迭代级调度
- 可插拔注意力后端与推测解码
- OpenAI/Anthropic 兼容 API
- 核心数据结构与内存管理
- 异步引擎与 ZMQ IPC 通信

## 如何使用本书

每个章节都包含**概述 → 原理 → 实践**三个层次，由浅入深。你可以按顺序系统学习，也可以直接跳到感兴趣的主题。
