# 大语言模型概述

大语言模型（Large Language Model, LLM）是基于 Transformer 架构、在海量文本上预训练的神经网络模型，具备强大的文本理解和生成能力。

## 发展历程

| 年份 | 里程碑 |
|------|--------|
| 2017 | Transformer 架构提出 |
| 2018 | GPT-1 / BERT 发布 |
| 2019 | GPT-2 展示零样本能力 |
| 2020 | GPT-3 175B 参数 |
| 2022 | ChatGPT 引爆 AI 革命 |
| 2023 | GPT-4 / LLaMA / Mistral |
| 2024 | 推理模型 o1 / 开源生态爆发 |
| 2025 | Agent 与多模态融合 |

## 核心概念

### 什么是语言模型？

语言模型的核心任务是预测序列中下一个 token 的概率：P(xt | x1, x2, ..., x(t-1))

### 涌现能力（Emergent Abilities）

当模型规模超过一定阈值时，会出现训练目标中未显式包含的能力：

- 上下文学习（In-Context Learning）
- 链式推理（Chain-of-Thought）
- 指令遵循（Instruction Following）

### 缩放定律（Scaling Laws）

模型性能与计算量、数据量、参数量之间存在幂律关系。这意味着在资源有限时，需要合理分配计算、数据和参数的预算。

## 主流模型分类

| 类型 | 代表 | 特点 |
|------|------|------|
| 闭源 | GPT-4, Claude, Gemini | 能力强，API 调用 |
| 开源 | LLaMA, Mistral, Qwen | 可本地部署，可定制 |
| 代码 | Codex, CodeLlama | 代码生成与理解 |
| 多模态 | GPT-4V, LLaVA | 图文混合理解 |
