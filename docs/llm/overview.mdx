# 大语言模型概述

大语言模型（Large Language Model, LLM）是基于 Transformer 架构、在海量文本上预训练的神经网络模型，具备强大的文本理解和生成能力。理解 LLM 需要先掌握一些机器学习的基础概念。

## 机器学习基础

机器学习是 AI 的核心子领域，旨在让计算机从数据中自动学习规律，而无需显式编程。

### 学习范式

| 范式 | 是否需要标签 | 典型任务 | 代表算法 |
|------|------------|---------|---------|
| 监督学习 | 是 | 分类、回归 | SVM、决策树、神经网络 |
| 无监督学习 | 否 | 聚类、降维 | K-Means、PCA、自编码器 |
| 半监督学习 | 部分 | 标签传播 | 标签传播、自训练 |
| 强化学习 | 奖励信号 | 序列决策 | Q-Learning、PPO |

### 核心概念

- **过拟合与欠拟合**：过拟合指模型在训练集上表现好但泛化差，欠拟合指模型连训练集都学不好
- **偏差-方差权衡**：泛化误差 = Bias² + Variance + Noise，好的模型需要在两者间找平衡
- **模型评估**：分类看精确率/召回率/F1，回归看 MSE/R²，交叉验证更充分利用有限数据
- **特征工程**：将原始数据转换为更适合模型的特征，包括标准化、编码、特征选择等

> LLM 本质上也是一个机器学习模型，只是规模更大、架构更复杂。上述概念在 LLM 中同样适用：预训练是无监督学习，SFT 是监督学习，RLHF 是强化学习。

## 核心概念

### 什么是语言模型？

语言模型的核心任务是预测序列中下一个 token 的概率：P(xt | x1, x2, ..., x(t-1))

### 涌现能力（Emergent Abilities）

当模型规模超过一定阈值时，会出现训练目标中未显式包含的能力：

- 上下文学习（In-Context Learning）
- 链式推理（Chain-of-Thought）
- 指令遵循（Instruction Following）

### 缩放定律（Scaling Laws）

模型性能与计算量、数据量、参数量之间存在幂律关系。这意味着在资源有限时，需要合理分配计算、数据和参数的预算。

## 主流模型分类

| 类型 | 代表 | 特点 |
|------|------|------|
| 闭源 | GPT-4, Claude, Gemini | 能力强，API 调用 |
| 开源 | LLaMA, Mistral, Qwen | 可本地部署，可定制 |
| 代码 | Codex, CodeLlama | 代码生成与理解 |
| 多模态 | GPT-4V, LLaVA | 图文混合理解 |
