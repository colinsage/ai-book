# 特征工程

特征工程（Feature Engineering）是将原始数据转换为更适合机器学习模型的特征的过程。好的特征往往比复杂的模型更重要。

> "特征工程是应用机器学习中最重要的技能。" — Andrew Ng

## 数据预处理

### 缺失值处理

```python
# 删除缺失值
df.dropna()

# 填充缺失值
df.fillna(df.mean())  # 均值填充
df.fillna(df.median())  # 中位数填充
```

### 数值特征标准化

```python
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# Z-score 标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Min-Max 归一化
scaler = MinMaxScaler()
X_normalized = scaler.fit_transform(X)
```

### 类别特征编码

```python
from sklearn.preprocessing import OneHotEncoder, LabelEncoder

# 独热编码
encoder = OneHotEncoder(sparse_output=False)
X_encoded = encoder.fit_transform(X_categorical)

# 标签编码
encoder = LabelEncoder()
y_encoded = encoder.fit_transform(y)
```

## 特征构造

### 数值特征变换

- 对数变换：处理长尾分布
- 多项式特征：捕获非线性关系
- 分箱（Binning）：将连续值离散化

### 时间特征

从时间戳中提取年、月、日、小时、星期几、是否节假日等。

### 文本特征

- TF-IDF
- 词嵌入（Word2Vec, BERT embeddings）
- 文本长度、特殊字符数量等统计特征

## 特征选择

### 过滤法

根据统计指标筛选特征：方差阈值、相关系数、互信息。

### 包裹法

以模型性能为标准选择特征子集：递归特征消除（RFE）。

### 嵌入法

利用正则化自动选择特征：L1 正则化（Lasso）、树模型的特征重要性。
