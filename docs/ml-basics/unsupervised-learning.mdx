# 无监督学习

无监督学习（Unsupervised Learning）处理没有标签的数据，目标是发现数据中的内在结构和模式。

## 聚类

聚类将数据点分组，使同组内的数据点彼此相似，不同组的数据点差异较大。

### K-Means

```python
from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)
```

**选择 K 值**：肘部法则（Elbow Method）和轮廓系数（Silhouette Score）是常用的方法。

### 层次聚类

通过计算样本间的距离，自底向上或自顶向下构建聚类树（树状图）。

### DBSCAN

基于密度的聚类方法，能够发现任意形状的簇并识别噪声点。

## 降维

降维将高维数据映射到低维空间，同时尽量保留重要信息。

### PCA（主成分分析）

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_reduced = pca.fit_transform(X)
```

### t-SNE / UMAP

用于数据可视化，能更好地保留数据的局部结构：

```python
from sklearn.manifold import TSNE

tsne = TSNE(n_components=2, perplexity=30)
X_embedded = tsne.fit_transform(X)
```

## 自编码器

基于神经网络的无监督学习方法，通过编码-解码结构学习数据的压缩表示，是深度学习中特征学习的基础。
