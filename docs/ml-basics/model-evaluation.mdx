# 模型评估

模型评估是机器学习中至关重要的环节，它告诉我们模型的表现如何，以及是否可以信赖。

## 数据划分

通常将数据分为三个集合：

- **训练集（Training Set）**：用于训练模型（约 60-80%）
- **验证集（Validation Set）**：用于调参和模型选择（约 10-20%）
- **测试集（Test Set）**：用于最终评估（约 10-20%）

### 交叉验证

K 折交叉验证（K-Fold CV）能更充分地利用有限数据：

```python
from sklearn.model_selection import cross_val_score

scores = cross_val_score(model, X, y, cv=5)
print(f"平均准确率: {scores.mean():.3f} ± {scores.std():.3f}")
```

## 分类指标

### 混淆矩阵

| | 预测正 | 预测负 |
|---|---|---|
| 实际正 | TP | FN |
| 实际负 | FP | TN |

### 常用指标

- **准确率（Accuracy）** = (TP + TN) / (TP + TN + FP + FN)
- **精确率（Precision）** = TP / (TP + FP)
- **召回率（Recall）** = TP / (TP + FN)
- **F1 分数** = 2 × (Precision × Recall) / (Precision + Recall)

### ROC 与 AUC

ROC 曲线描绘了不同阈值下的 TPR-FPR 关系，AUC 值越大表示模型越好。

## 回归指标

- **MSE（均方误差）**：(1/n)Σ(y_i - ŷ_i)²
- **MAE（平均绝对误差）**：(1/n)Σ|y_i - ŷ_i|
- **R²（决定系数）**：模型解释的方差比例

## 超参数调优

```python
from sklearn.model_selection import GridSearchCV

param_grid = {'C': [0.1, 1, 10], 'kernel': ['linear', 'rbf']}
grid_search = GridSearchCV(SVC(), param_grid, cv=5)
grid_search.fit(X_train, y_train)
```
