# 机器学习概述

机器学习（Machine Learning）是人工智能的核心子领域，旨在让计算机从数据中自动学习规律，而无需显式编程。

## 什么是机器学习？

> "机器学习是让计算机在没有明确编程的情况下能够学习的科学。" — Arthur Samuel (1959)

机器学习的基本流程：

1. **数据收集** — 获取与任务相关的数据
2. **特征工程** — 从原始数据中提取有效特征
3. **模型选择** — 选择适合任务的算法
4. **训练** — 用数据拟合模型参数
5. **评估** — 在测试集上衡量模型表现
6. **部署** — 将模型应用于实际场景

## 主要学习范式

| 范式 | 是否需要标签 | 典型任务 | 代表算法 |
|------|------------|---------|---------|
| 监督学习 | 是 | 分类、回归 | SVM、决策树、神经网络 |
| 无监督学习 | 否 | 聚类、降维 | K-Means、PCA、自编码器 |
| 半监督学习 | 部分 | 标签传播 | 标签传播、自训练 |
| 强化学习 | 奖励信号 | 序列决策 | Q-Learning、PPO |
