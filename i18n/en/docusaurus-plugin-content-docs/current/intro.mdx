# AI Book — From Basics to Practice, Master AI Systematically

Welcome to AI Book! This is a systematic AI knowledge base covering four major topics:

- **Machine Learning Basics** — Supervised learning, unsupervised learning, model evaluation, and feature engineering
- **Large Language Models** — Transformer architecture, pre-training & fine-tuning, inference optimization
- **Prompt Engineering** — Chain-of-Thought, Few-shot, Agent prompt patterns
- **AI Application Development** — RAG, LangChain, AI Agent development in practice

## How to Use This Book

This book follows a progressive structure. We recommend reading chapters in order. Each topic includes:

1. **Overview** — Quickly grasp the core concepts of the field
2. **Principles** — Deep dive into the technical details
3. **Practice** — Hands-on exercises with examples and code

Select a topic from the left navigation bar to start your learning journey!
